Ycd Bank – Modern Banking UI

A clean and responsive front-end web interface for a digital banking application.
This project showcases a modern UI/UX for viewing account information, managing transactions, and navigating a banking dashboard.

Features:

Modern and clean interface

Responsive design for different screen sizes

Reusable components (buttons, cards, sections)

Organized folder structure (images, styles, scripts)

Fast and lightweight — built using only HTML, CSS, and JavaScript
